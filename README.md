VisuAlive + VCCW + Gulp (vavg)
=========================

簡単なコマンドを実行するだけで、WordPress の開発環境 ( VCCW ) と Gulp ( Scss のビルドや BrowserSync など ) 環境を整えることができます。  
あらかじめ、Ruby、Vagrant、Node.js、Sass、Compass、ImageMagic をインストールを完了しておく必要があります。

## How to use
### Install
```
$ chmod u+x ./vavg
$ ./vavg -i
```

### Start
```
$ ./vavg -s
```

### Install & Start
```
$ ./vavg -i -s
```

## Support OS
* OSX: 10.10.5
* Windows: not tested
